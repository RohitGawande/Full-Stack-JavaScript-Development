Introduction to Higher-Order Functions and Callbacks

What Are Higher-Order Functions?

A Higher-Order Function (HOF) is a function that either:

  • Accepts other functions as arguments (callbacks)
  • Returns a function as its result

They help make code more abstract, reusable, and easier to manage.

Callback Functions

A callback function is a function passed as an argument to another function and executed later.

This pattern is essential in JavaScript for asynchronous operations and cleaner code logic.

Example: Cube Calculation using Callback

const callback = (n) => {
  return n ** 2;
};

function cube(callback, n) {
  return callback(n) * n;
}

console.log(cube(callback, 2)); // Output: 8

Why Use Higher-Order Functions?

  • Code Abstraction: Encapsulates repetitive logic, improving code reusability.
  • Cleaner Code: Callbacks maintain logical separation, following the DRY principle.
  • Asynchronous Programming: Essential for non-blocking operations in JavaScript.

Real-World Example: Asynchronous Callback

Using setTimeout to demonstrate asynchronous callbacks:

setTimeout(() => {
  console.log("Hello, Rohit!"); 
}, 2000);

This code logs Hello, Rohit! to the console after 2 seconds.
